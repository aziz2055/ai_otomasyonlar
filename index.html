Token hesaplama ve yapay zeka hizmetlerimiz hakkında merak ettiğiniz her şey

Token nedir ve nasıl hesaplanır?

Token, yapay zeka modellerinin metni işlemek için kullandığı temel birimdir. Kelimeler, alt kelime parçaları veya tek karakterlerden oluşabilir.

Sistemimizde her karakter 0.4 token olarak hesaplanmaktadır. Bu nedenle 10 karakterlik bir metin yaklaşık 4 token'a eşittir.

Noktalama işaretleri de token sayısına dahil edilir.

Neden token sayısı önemlidir?

Token sayısı, yapay zeka modellerinin işleyebileceği metin uzunluğunu belirler. Modellerin genellikle maksimum token sınırı vardır.

Ayrıca, birçok API hizmeti kullanım başına token sayısına göre fiyatlandırma yapar, bu nedenle token sayısını bilmek maliyet tahmini yapmanıza yardımcı olur.
